# AI Multi-Agent Backend

This is the backend structure for an AI multi-agent workflow system built with FastAPI.
It includes API routes, models, services, and database structure.

## Features
- FastAPI
- SQLAlchemy ORM
- PostgreSQL
- CRUD APIs
- Modular architecture

## Run the Server
uvicorn app.main:app --reload
