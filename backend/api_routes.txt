# Core FastAPI routes (Sonic AI)

## Conversations

POST   /conversations/
GET    /conversations/
GET    /conversations/{conversation_id}
GET    /conversations/{conversation_id}/messages
PATCH  /conversations/{conversation_id}

## Projects

POST   /projects/?owner_id={user_id}
GET    /projects/
GET    /projects/{project_id}

## Tasks

POST   /tasks/
GET    /tasks/
GET    /tasks/{task_id}

## Agent logs

POST   /logs/
GET    /logs
GET    /logs/{log_id}
GET    /logs/task/{task_id}
DELETE /logs/{log_id}
